explanations
What does "max volumetric flow" actually mean?

It's the maximum amount of plastic your extruder can push per second while still maintaining grip on the filament. Above this value the motor either slips on the gear teeth or extrudes filament that wasn't fully melted. Both lead to under-extrusion in real prints. Slicers use this value to cap volumetric speed even when you set high print speeds.

StallGuard, SG_RESULT, CV — what's measured?

StallGuard is a TMC driver feature that measures motor torque without external sensors. It works by analyzing the back-EMF of the coils. SG_RESULT is its output — on TMC5160 it ranges 0–1023 (lower = more load), on TMC2209 it's 0–510. The plugin samples this value many times per second.

Run-to-run CV is the coefficient of variation between repeated measurements at the same flow. Low CV (<3 %) means measurements are repeatable. High CV means something is inconsistent — usually slip starting.

Why the 80% / 90% recommendations?

The "Maximum Safe" value is what the plugin found just below where slip starts. The 80% recommendation gives you a 20% safety buffer — recommended for everyday use because filament thickness varies, hotend temperature drifts, and longer prints stress the system more. The 90% value is closer to the limit and should only be used after you've validated it with several long real-world prints.

Hotend temperature & heater PWM

The thermal chart logs the hotend's behavior during each step. Temperature drops at high flow happen when the heater can't melt filament fast enough. Heater PWM reaching 1.0 means Klipper is asking for full power; if it stays there for several seconds AND temperature falls, the system is near thermal saturation. Driver thermal flags (OTPW ≥120 °C, OT ≥150 °C) appear as triangles when the TMC chip itself gets hot. None of this triggers the test or affects slip detection — purely informational.

Residence time in the melt zone

The number of seconds each piece of filament spends inside the heated melt zone. Computed as melt_zone_length / linear_feed_speed. As flow increases, residence time shrinks — even with plenty of heater power, very short residence times mean the filament core stays cold.

Each hotend design has a minimum residence time needed for clean melting. The reference lines in the chart show these minimums, ordered from weakest to strongest:

  • V6-class — minimum ~1.5 s residence
  • Volcano-class — minimum ~0.6 s residence
  • CHT / HF-class — minimum ~0.3 s residence
  • Goliath / Bondtech-CHT — < 0.3 s ok

The tooltip shows the minimum required hotend class at each flow — anything stronger also works. Example: 0.8 s residence needs at minimum a Volcano, but a CHT or Goliath would also handle it fine. If your hotend is below the minimum class shown at the slip point, the bottleneck was filament speed (not heater wattage).

Coloured zones & thermal stress score

The chart backgrounds are tinted in three zones based on a heuristic thermal stress score calculated per step. The score combines five signals (each weighted differently):

  • Heater PWM level — how hard the heater is working (0–30 points). PWM 30% adds nothing, 95%+ adds full 30.
  • Temperature drop from target — how much actual temp falls below target (0–25 points). 0 °C drops adds nothing, 5 °C+ adds full 25.
  • PWM rising trend — heater taking on more work relative to early-test baseline (0–15 points).
  • PWM peak hits — saturation events even briefly (0–10 points).
  • Intra-run SG drift — load drifting upward DURING individual 5-second runs (0–20 points). This is a strong cold-extrusion signature: as filament heats up too slowly, motor load increases continuously through a single run rather than staying steady. Pure motor slip jumps abruptly; cold extrusion drifts gradually.

Score interpretation: 0–30 = green / stable, 30–60 = yellow / moderate stress, 60–100 = red / high stress (cold extrusion likely).

The yellow zone in the charts marks where the score first crosses 30. Important: This is a heuristic combining heater behavior with within-run load trends. It does NOT trigger or modify the slip-detection logic. Hover any thermal-chart point to see the per-step score AND the intra-run drift value (positive % = load increased during the run, the cold-extrusion fingerprint).

About the data: All thermal values (PWM, temperature) are sampled at 4 Hz during active extrusion only — never during the cooldown gaps between reps. This avoids the otherwise confusing pattern where PWM drops to baseline and temperature overshoots while the heater is idle.
